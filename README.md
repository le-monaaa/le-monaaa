# 🚀 도전하는 개발자 이유빈입니다

사용자의 요구를 깊이 이해하고, 안정적이면서도 효율적인 시스템을 설계하는 데 주력하고 있습니다.

이러한 태도로 개발자로서 한 단계 더 성장하며 의미 있는 변화를 만들어가고자 합니다.
